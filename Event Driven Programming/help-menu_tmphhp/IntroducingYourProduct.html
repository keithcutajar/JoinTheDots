In this program one is asked to enter his Name, Username and Password.

When he clicks  on the start button and the inputs are valid, he is redirected into the main page of the program where he can choose a number from 2 to 15.

When a number is chosen, a number of dots according to the number chosen are shown at a random spot and with a random color.

You can draw by clicking and moving the mouse. One can change the color of the drawing like by clicking on the "Join The Dots!" button.

If the Textbox on the top-right is filled, and the button with the test "Save"  on it is pressed, a window to save the document is popped.

When the link in the bottom right corner is clicked, a window is popped so that the user can load an image.
