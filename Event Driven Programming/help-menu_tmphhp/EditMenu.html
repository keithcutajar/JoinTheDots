Below one can find the Main Page.


When one clicks on the dropdown menu, one can choose a number between 2 and 15.
When a number is chosen, a number of dots will be displayed  on the screen at a random point with a random color.




When the button "Join the Dots" is clicked, a palette is popped out so that the user can choose a color to draw with.
